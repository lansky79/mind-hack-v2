# 04 - 用数学公式穿越时空

我的创业经历有成功的也有失败的。从 2012 年，从营业额月数十万到GMV月数亿我都做过。

若要我总结这八年来的最重要一条宝贵经验。就是创业不是靠运气，而是靠设计。有时候甚至是一条数学公式就能看终局。

有些点子与公司，从数学上就能看终局。根本不需要白费力气。

## 用逆推法 + 数学公式预先看终局

2020 年，因为 COVID-19，网路开课平台很热。于是过去一些在台湾担任老师的朋友甚至是网路圈未接触教学的朋友，都认为线上教学是个风口。磨拳擦掌的想跳入开课平台这个行业。还有人问我是否愿意跟他合伙或者是投资他的平台。或者有什么增长战略可以建议。

我对这个点子，完全不看好，甚至还劝阻他们投入。让他们觉得很诧异。

为什么我如此不看好呢？

其实，如果这个品类，「在台湾」只有一间公司做，而且是几年之前做，还有机会。但是如果是2020年有10-20 间公司作，那就根本没做的必要了。

原因是在台湾，这不是个好生意（注），不太能增长、扩张。甚至跟创始人能力资源无关。

在我做过那么多公司之后，我总结出了一个业绩公式：

> 业绩 = 有此需求用户数 x 消费次数 x 频率 x 裂变数

### 拆解开课平台利润的数学公式

好，为什么开课平台不是一个好生意呢？

我们来看这个开课平台，我们需要哪些人来参与这个活动。

第一个，开课平台它是一个双边供给的生意。也就是说开课平台必须要两种人的参与，一种是老师的参与，一种是学生的参与。

要做开课平台，你得先要有老师。而这些老师也要有办法产出课，所以这是一个蛮严重的问题。双边的生意其实并不好做，如果你打算创业，尽量是做单边生意。比如说你开课，就是单边的，只要搞定学生方面的获取渠道就行了。

但是，如果你要做一个开课平台。你就得

1. 找到老师
2. 帮这些老师产出课
3. 帮老师招到学生

这是一个考验双边的supply的问题。任何一间业者的运营成本都会非常大。也完全超过一般创业者的能力。

有些人可能会觉得开课利润挺大，自然开课平台利润更大。所以有些人不仅当老师，后续还想去做开课平台。

因为，开课平台，本身可能可以抽的利润比例挺高。20%-50%都有可能，创业者会觉得这生意能做。

但是这背后隐含着运营的成本极其之大。

如果你只是一个程式设计师，我完全不建议你做一个开课平台。因为这个idea会有很大量的营运成本，而这成本完全不是没有教学相关资源的人可以投入。如果今天是一个管顾，手上握有大量老师或者大量学生资源，这个生意可能可以做。因为他能搞定老师的supply，也能搞定学生的supply。

管顾去做这件事情，反而可能是有点机会的。

注意，我说是有点机会。

但是如果你只是一个草根的创业者，去做这件事情是有难度的。但也不是说不能做，那看你什么时候做。比如说就在很前面的时间去切入做这件事情。大陆很红的小鹅通，其实是在风口的很前面的时候就切入这个领域。事实上是吴晓波，因为要讲课，吴晓波的粉丝有几十万粉丝，所以他就请了这个小鹅通创办人，很厉害的 RD，接案做了这个上课系统。

这个案子结束之后，创业者去把这个东西做了平台化，投放给全国的老师跟用户去做。小鹅通因为是非常的早期产品，技术能力又强，几乎没有人跟他抢。后续因为中国的人口数量太大了，也有覆盖几乎全中国的微信网路（小鹅通是基于微信渠道）。所以基本上没什么老师、学生供给上的问题。

另外，我之所以讲开课平台这个点子在台湾有问题。是因为在大陆问题比较小。中国的教学市场，天花板非常的高，因为人口有14亿，数量实在太多了。任何细分领域下去做都可以产生正循环。不用说学生了，就连老师都可能有成千上万，甚至上10。

你不用担心老师到底有多少人。但是像开课平台这件事情，那台湾的老师有多少人？

这就是一个很吊诡、很吊诡的事情了，因为老师要具备很多，他要有专业技术，他也要能讲课做课。

有些老师是有技术，但没有能力开课的。所以你选择做开课平台，你是不是要去帮他搞定做课的事呢？再来，老师有没有自带粉丝。因为有些老师就是个素人，虽然技术很强，但没有粉丝，在商业上这是个严重问题，因为你要帮他教育市场。

台湾的根据我自身熟悉的，大概上得了台面，自己有粉丝，自己有课，自己又能讲的，应该不超过100个。

我是认真地说，就是这些你能叫得上名号，可以开公开班，又有吸粉能力的，应该不超过100个。

那不超过100个，它会有一个严重的问题。为什么呢？

假设这100个老师都来你这个平台，我们就直接不讲竞争的事情。

假设这100个老师都来用你的平台，他一次的课，比较贵的课可能就是说用户是100人，学生是100个人，学费大概收2000~3000，我们算2000好了。

好，100乘以100再乘以2000是多少？2000万。OK，那总业绩是营收是2000万有点太少了。

因为如果一年营收是2000万，你要注意，这2000万还要扣掉你的制作成本，还要扣掉你的税跟老师的分红，跟营运成本之类的，那净利是多少？如果净利是10%，这件事你一年只赚200万而已，甚至是会更少。你会不会发现这个数字有点低得恐怖？

可能你感觉不可能这么低吧。我们把学生调到 300人。那么这一年营收就是 6000万。

这样算下来一年的净利是600万。这 600 万值得赚吗？

坦白说我当年没有开教学平台，只当老师，光教增长与编程的线下班，利润就超过 600 万了。

但我甚至不需要这么费力。

### 成人教学市场没有增长性

回来谈学生供给与成长问题。

其实，课这个品类，有一个问题。就是课越卖，学生越少。

为什么？因为就是课本上是一个改变用户状态的产品，你上完这个课就不会再重复上了。不管你再喜欢这个老师，你都不会再上了。所以你一年之内，会上这个老师多少堂课呢？可能只能一场而已。一生，甚至上这个老师，顶多三堂（初级高级中级）而已。

因为，老师也不可能开那么多堂课。是不是觉得这行业成长有限了。

那假设我们最好的情况，一个学生一年上老师三堂课。那就是乘以3了。6000乘以3，那就是1.8亿。

这是最好的情况下，所有的老师、所有的学生都在你这里上课。撑死了就是1.8亿。

再来，1.8 亿，是每年，还是lifetime？

是我今年只能做这一次生意，还是明年可以再做？你要注意，学完了三堂课以后，你基本上很难再对一个老师产生再上课的兴趣了。

如果你针对的又是成人教育市场的话，这个营收就是一次性的....。

有没有发现这个东西好像是个很大的坑。

所有的厂商来竞争这只有1.8亿的市场，这值得吗？

### 资本不可累积，不进反退

再来，还不是最坑的问题。本来，开课平台，它最大的瓶颈点，就是老师的增长。

理论上是不是增加老师，就能增加学生，对吧？

好，那增加老师简单吗？其实并不简单。

为什么呢？因为开课是门技术，不是所有的老师都真的能开课，你帮他辅导这一次做课做完以后，他会不会愿意在你这个平台上面。unlikely。

为什么呢？因为有些老师从不会开课到会开课，可能只是和你合作一次，被平台高抽成，是他第一次的学费。下次，这个老师可能去其他间再开。因为其他地方平台，有更好的比例，不是吗？

结果，原先很大的运营成本投入，并不能重复再利用。你培养一个老师，但这个老师的寿命只有做一档的机会。可能跟这个老师的 Life Time revene，就只有一次合作机会。

### 不可裂变

再来有下一个这个问题，就是说能力跟意愿的问题。来上课的学生，看到开课很好赚，想要变成老师，那这个件事情有没有可能呢？

可能也是unlikely。

为什么？这些学生有意愿当老师，去收智商税。但是这些学生没有能力。

所以这个平台裂变是低的。因为被卡在意愿跟能力的问题。

一个老师可以收100个学生，这100个学生里面未必出得了一个老师。就是一个学生能够变成老师再上来开课，你会发现这件事情有点恐怖，是不是？你必需要投入很大的资本才能成长。

### 拆开开课系统结构

既然课的坑这么多，成本又如此之大。那我们是不是可以退一不再去看一件事情。

假设这个系统上面不卖课，只卖课的门票，那可不可以做？

好像可以，对吧？

卖课系统其实是 content 加 ticket system，对吧？既然课有 unscalble 的问题。

我们为何不只做课的 ticket system。

课的 Ticket system赚不赚钱？我跟你说，不赚钱。

为什么呢？如果你没进过讲师界，就不知道一个秘密。很多老师开课是不用 ticket system，很多老师至今还在用转账的方式。

因为老师不想被扣税，不想要被平台抽手续费，也不想要让国税局知道他赚钱，所以都用转账的方式收学费。一个 ticket system 收 5%。

如果这个老师课是高级课，卖 15000以上。平台就抽 700。所以这些老师是不会用你的ticket system，无论它多好用。

所以这行其实是一个很吊诡的现象了，你的供应商是没有动机去用你的系统并且跟你长久做生意的，所以它是一个很容易、很脆弱的结构。

在中国大陆为什么可以做得很大，是因为他们有非常多的老师，非常多的素人，非常多的学生。加上平台池子非常大，所以这个结构并不
会那么的脆弱。不太存在鱼被捞完的这件事情，老师用完或学生用完。

但是台湾的市场太浅了，是没有办法做这件事情的。

## Ticket system 有更好的模式

再来，ticket system是不太赚钱的，为什么？

刚刚说道有些老师不太愿意上这个ticket system的原因，是因为他一张票的价格，比如说他一个课要卖15000，ticket system，你要抽10%就是1500，抽5%就是750，他不觉得你值750。

所以你就得要冲量，因为这5%甚至不是你全收的，可能2%、3%是要给金融机构的，不是你全收的。所以它就并不是那么赚钱，ticket system 如果真的要赚钱，你一定要做演唱会票的这个系统，万人大会这样子。

这样座这个平台，才会得到最大的效益。

再来，台湾的ticket system系统能赚得钱也是有限的，所以它本身是一个很难长大的品类。

所以你自费去创这个业，甚至你投资做这件事情，我都并不觉得它是一个简单的题目。

不要说好跟不好，但它其实不会是一个简单的生意，你可以再去做其他更简单的事情，你为什么要来做一个地狱难度的事情呢？

因为你会觉得看到别人在赚钱，你也想赚钱，你也觉得这个东西很大。但是其实在这个模型上面，我们用数学一算，你会发现它其实并不是一个会赚钱、会成长的生意，那你有没有必要去做呢？

当然有些人想做，也已经做下去了，那怎么办？

你可以再去想说，如果我做了一个上课的系统，这个系统我可以卖给谁，你未必要做一个给成人用的开课系统。如果你做的是一个大众开课的班，这可能没有市场。但是你如果说做一个企业版，或是你做K12这些开课的系统，你会想说也许你可以做看看。

为什么呢？因为比如说是教育的这个系统，你看K12吧，如果说你是英语的这个系统，K12跟英语教育其实是挺好的教育体系。

你知道为什么吗？K12就是6~18岁的人，这些人学什么东西隔年就忘掉，瞬间就忘掉，所以他并不会改变他的状态，有没有？

英语跟日语你学完之后会忘掉，所以为什么上市公司基本上都是做K12、做教育的。

你看这边每年的生意市场会有多大。

所以我其实是要说，不需要看到一窝风就冲进去。倒过来+用数字化拆解，你也许就知道这个生意有多大，能不能做。要不要跳进去抢。

不需要做到一半的时候才发现钱烧光了，找不到出路，你时间去调研市场。

这种挺地狱开局的模式，你在纸上写写算术就会知道，也许我们不需要花我们的精力去投资这个事业。
